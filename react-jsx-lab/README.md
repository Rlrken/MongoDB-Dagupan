# React JSX Lab

This project demonstrates basic React JSX concepts.

Features:
- Functional React components
- JSX rendering
- Dynamic data using variables
- Rendering lists using map()
- Reusable components with props

- Created React project structure
- Configured Babel
- Rendered JSX to DOM
- Added dynamic JSX content
- Used map() for rendering lists
- Created reusable Item component
