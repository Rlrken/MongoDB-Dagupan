# JavaScript Conditional Statements, Loops, and Functions

## Overview

This lab demonstrates:
- Conditional statements
- Loops
- Functions
- Terminal execution using Node.js

---

# Step 1: Project Setup

A project folder was created containing:
- script.js
- README.md

The project was initialized using:

```bash
npm init -y
```

---

# Step 2: Conditional Statements

Conditional statements were implemented using:

- if
- else if
- else

Example:

```javascript
if (score >= 90)
```

Purpose:
- Execute different outputs based on conditions.

---

# Step 3: Loops

Loop examples included:

- for loop
- while loop

Purpose:
- Generate repeated output automatically.

Example:

```javascript
for (let i = 1; i <= 5; i++)
```

---

# Step 4: Functions

Reusable functions were created to:
- Display greetings
- Perform addition

Example:

```javascript
function addNumbers(a, b)
```

Purpose:
- Avoid repeated code
- Improve code organization

---

# Step 5: Functional Testing

The application was executed using:

```bash
node script.js
```

The following features worked successfully:
- Conditional logic
- Loops
- Functions
- Console outputs
