# Responsive Web Pages Using CSS Media Queries

## Overview

This project demonstrates how to create a responsive web page using CSS Media Queries.

The layout automatically adjusts based on different screen sizes such as:
- Desktop
- Tablet
- Mobile devices

---

# Technologies Used

- HTML5
- CSS3
- JavaScript

---

# Step 1: Project Setup

The following files were created:

- index.html
- style.css
- script.js

---

# Step 2: HTML Structure

The webpage contains:
- A page title
- Two div containers
- Class names for styling

Example:

```html
<div class="box box1"></div>
<div class="box box2"></div>
```

---

# Step 3: CSS Styling

CSS class selectors were used to:
- Add background colors
- Increase font size
- Make text bold
- Add spacing and layout styling

---

# Step 4: Media Queries

Media queries were used to create responsive layouts.

## Tablet Breakpoint

```css
@media (max-width: 768px)
```

Changes:
- Boxes stack vertically
- Font sizes adjust

---

## Mobile Breakpoint

```css
@media (max-width: 480px)
```

Changes:
- Smaller padding
- Smaller font size
- Mobile-friendly layout

---

# Step 5: Responsive Testing

The webpage was tested using browser developer tools.

Tested screen sizes:
- Desktop
- Tablet
- Mobile

The layout adjusted correctly without overlapping elements.
