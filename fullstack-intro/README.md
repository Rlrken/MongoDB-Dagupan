Introduction to Full Stack Development & Web Architecture (MERN Overview)
Client (Browser)
	It is the application or device that initiates a request for services, data, or resources from the server. It is the interface for users to manage inputs and display results returned by the server.
Frontend (UI Layer)
	It is part of the website or application that users see. It is important for the Frontend or UI layer to be user-friendly as it includes everything from the layout, colors, fonts, to buttons, animations, etc. Frontend Development uses HTML, CSS, and JavaScript.
Backend (Server)
	It is the server-side component of a website or application that operates behind the frontend. It provides functionality that users do not directly see. It also manages data storage, server logic, and API interactions.
Database (MongoDB)
	It is an organized, electronic collection of structured or unstructured data designed for efficient storage, retrieval, modification, and analysis. An example of a database is MongoDB, which is a leading NoSQL document database designed for scalability and flexibility.
